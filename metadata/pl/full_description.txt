Cechy:
- tłumacz swój tekst nawet na 16 języków
- 5 różnych motywów
- łatwo udostępniaj tłumaczenie z aplikacji
- całkowicie otwartoźródłowy
- więcej wkrótce...

Obsługiwane języki (UI):
- angielski
- niemiecki

Obsługiwane języki (tłumaczenie):
- angielski
- arabski
- chiński
- francuski
- hindi
- hiszpański
- indonezyjski
- irlandzki
- japoński
- koreański
- niemiecki
- polski
- portugalski
- rosyjski
- turecki
- wietnamski
- włoski

Tłumaczenie:
Do tłumaczenia jest używany interfejs API <a href="https://libretranslate.de/">LibreTranslate</a>. LibreTranslate jest objęty licencją <a href="https://www.gnu.org/licenses/agpl-3.0.en.html">AGPLv3</a>.
